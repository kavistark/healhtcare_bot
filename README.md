# Mosaic – AI Healthcare Provider Engagement Assistant

Mosaic is an AI-powered Healthcare Provider (HCP) Engagement Assistant designed for pharmaceutical sales and field teams. It enables users to interact with healthcare provider data using natural language, eliminating the need to manually analyze spreadsheets.

The application combines **Django**, **Google Gemini AI**, **Machine Learning**, and **predictive analytics** to deliver intelligent insights, doctor prioritization, KPI summaries, and prescription forecasting through an easy-to-use chatbot interface.

## Features

* 🤖 AI-powered chatbot using Google Gemini
* 📊 Healthcare Provider (HCP) data analysis
* 🎯 Doctor priority scoring
* 📈 Prescription prediction using Machine Learning (Random Forest)
* 📋 KPI dashboards and summaries
* 🔍 Advanced filtering by territory, channel, patient type, and more
* 📄 Export chat responses as Markdown reports
* 🔄 Intelligent fallback logic when AI services are unavailable
* 🌐 Django web application with responsive chat interface
